-----------------------------------------------Instruções-----------------------------------------------------

Para melhor uso das funções copie a pasta "Function and tools" e cole na pasta "toolbox" do MATLAB

Diretório de instalação\MATLAB\toolbox

O diretório de instalação padrão do MATLAB é:
C:\Arquivos de Programas\MATLAB\toolbox

Em seguida abra o MATLAB, clique na aba "Home" e clique em Set Path
Clique em Add Path -> Selecione a pasta "C:\Arquivos de Programas\MATLAB\toolbox\Function and tools" -> Save -> reinicie o MATLAB

Agora as funções e ferramentas podem ser utilizada sem que as arquivos estejam na mesma pasta do seu trabalho.


-----------------------------------------------Instructions-----------------------------------------------------

For better use of functions and tools, copy the folder "Function and tools" and paste to the "toolbox" folder of MATLAB

Instalation folder\MATLAB\toolbox

The standard MATLAB instalation folder is:
C:\Program Files\MATLAB\toolbox

Then open MATLAB, select tab "Home" and click in Set Path
Click in Add Path -> Select the folder "C:\Program Files\MATLAB\toolbox\Function and tools" -> Click in Save -> restart MATLAB

Thus, the functions and tools can be used without the necessity to place the files in your working folder.
